SKILL REFERENCE

Task Bar Hero skills: evaluate the effect in the activity you play.

A skill is not strong in the abstract. Its value depends on target count, timing, cooldown reliability, gear context and the stage where it is used.

FIRST FILTER

Ask whether the skill improves your present wall: repeated waves, a single target, or survival. Do not compare effects without matching the target activity.

Skill evaluation checklist

  • Does it affect the target that is stopping progress?
  • Can it be used reliably inside the run’s timing?
  • Does its effect change after a gear or Rune choice?
  • Was the description checked against the current version?

Connect skills to a route

BuildsGive each skill a clear job.RunesSupport the same bottleneck.StagesTest against a real encounter.

Named skill records will be added with current-game evidence rather than copied descriptions of unknown age.
