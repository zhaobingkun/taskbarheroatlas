RUNE ROUTE / SURVIVAL

Task Bar Hero survival Runes: stabilize the actual failure point.

Use a survival route when the run ends before extra damage or speed can become useful. Stability is a progression stat when it turns a failed route into a repeatable one.

WHEN TO USE IT

Choose survival when the same point ends multiple runs. Once the loop is stable, test whether the next bottleneck has moved back to damage or clear speed.

Stability test

  1. Identify the point where the run consistently ends.
  2. Change only the defensive route relevant to that failure.
  3. Compare completion rate across the same stage.
  4. Return to offense when stable completions expose a new wall.
