Four-step route test

  1. Name the bottleneck. Record whether the loss is wave time, a boss, or survival.
  2. Use one fixed target. Compare against the same stage and build conditions, not a different run.
  3. Change one meaningful input. Move a Rune choice while keeping gear, party and target stable where possible.
  4. Save the result with the patch. Keep clear time, completion rate and version beside the observation.

What is confirmed about the game scope

The official Steam page describes TBH: Task Bar Hero as a taskbar idle RPG with classes, skills, items and builds across three Acts and four difficulty tiers. That confirms the wider progression context; it does not confirm an individual Rune's value or a single best route. The distinction is why Atlas keeps current node data separate from route methodology.

Rune FAQ

What is the best Task Bar Hero Rune route?

There is no responsible universal answer without a version, a target activity and a stated failure point. A farming route can be wrong for a boss or unstable survival run.

Why are there no node values on this page yet?

Atlas adds a number only when it has a current source or a repeatable test record. This prevents an old image or copied list from becoming false database data.
