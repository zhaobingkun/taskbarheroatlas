Task Bar Hero chests: source and timing need context.

A chest result is only useful evidence when its source, event conditions, version and observation count are visible alongside it.

DATA BOUNDARY

This page does not claim a universal chest schedule or reward table. It is the child page for verified source notes and clearly labeled observations.

Record a chest observation properly

  • Name the chest or source exactly.
  • Include the active version and any event condition.
  • Count attempts, not only desirable rewards.
  • Mark the result observed until current evidence supports it.
