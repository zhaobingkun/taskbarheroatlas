# Task Bar Hero 项目说明

## 项目背景

- 研究与维护 Task Bar Hero 主题网站。
- 当前重点站点：`https://taskbarhero.wiki/` 与 `https://taskbarhero.org/`。

## 目标与边界

- 目标：建设可长期扩展、可抓取、对玩家有真实帮助的 SEO 内容与数据库站。
- 优先覆盖数据库、工具、攻略、版本更新、长尾问题和多语言入口。
- 未经明确要求，不发布内容、不修改线上站点、不执行账号或外部平台写操作。

## 工作规则

- 每次开始工作先读取本文件和 `memory.md`。
- 每次工作结束后把关键结论、变更、验证结果和踩坑补充到 `memory.md`。
- 网站遵循项目提供的 SEO 建站执行手册：原始 HTML 可抓取、唯一 title/meta/H1、正确 canonical、robots、sitemap、结构化数据、Hub/子页内链及图片 alt。
- 优先以官方游戏版本与可验证数据为准，版本号、更新时间和数据库统计必须保持一致。
