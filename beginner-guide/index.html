Task Bar Hero beginner guide: build one stable loop first.

Early progress is easier when each part of the run has a job: clear routine waves, pass the next hard check, and remain stable enough to repeat the route.

Open beginner build notes →
FIRST HOUR RULE

Do not chase every system at once. Find the first stage that does not clear reliably, name the reason, and make one targeted change before moving on.

A simple first route

  1. Choose a stable stage or loop you can repeat.
  2. Identify whether time, damage or survival is the first limiter.
  3. Assign your build a clear role for that limiter.
  4. Use Rune points to support the same job, then test again.

Beginner FAQ

What should a new player optimize first?

Optimize reliability on the next route. A stable clear gives better information and more useful progress than a route that only works occasionally.
