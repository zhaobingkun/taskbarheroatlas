Task Bar Hero drop rates: label the evidence before the percentage.

A useful drop page separates confirmed sources from player observations and unknown values. It should never make a percentage look official when the data is not verified.

SHORT ANSWER

No universal numeric drop table is published on this first release. Use the status label to understand whether a source is confirmed, observed, or waiting for verification.

How the drop desk will label data

LABELWHAT IT MEANSHOW TO USE IT
ConfirmedDirectly supported by current official or in-game evidence.Safe to use as a source reference.
ObservedPlayer-reported pattern with a stated sample and version.Useful lead, not a guaranteed percentage.
UnknownNo sufficient current evidence yet.Do not use as a farming promise.

Why a transparent unknown is better than a fake rate

Drop-rate pages attract strong search intent because players are deciding where to spend their next session. Made-up percentages may gain clicks but lead to bad routes and make later corrections harder. This URL is ready for versioned tables once data is collected with a source, date, sample context, and an explanation of what can change between patches.

How to record a useful observation

  1. State the game version and the exact source being tested.
  2. Count attempts as well as successful drops; a lucky streak is not a rate.
  3. Keep chest, stage, event, and difficulty conditions separate.
  4. Share the sample size and mark the result as observed until it is independently verified.

Drop Rates FAQ

Are the numbers on this page official?

Only entries explicitly marked Confirmed should be treated as supported by current evidence. This first release does not publish invented numeric rates.

Can a player report become a confirmed rate?

It remains an observation unless it is supported by verifiable current evidence. More samples can improve confidence without turning a community estimate into an official value.
