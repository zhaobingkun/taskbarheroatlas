Start with three answers

What ends the run?Wave time, a hard target, or a survival failure?
What stays fixed?Use the same stage and build for the comparison.
What changes once?Move a relevant point before changing unrelated gear.
What proves it?Clear time, completion rate or an exact failure point.

Rune Tree FAQ

Is there one best Rune Tree?

No. The best path changes when the active bottleneck changes. A screenshot without a version, stage and result is not enough context to copy.

When should I reset or change a route?

Revisit it after a patch, a meaningful gear change, a new stage wall or a measured result that shows the previous goal is solved.
