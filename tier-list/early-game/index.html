TIER LIST / EARLY GAME

Task Bar Hero early-game tier list: rank what you can actually use.

Early options should be evaluated by accessibility and the first progression wall, not by an endgame ranking with missing gear and systems.

RANKING RULE

An early-game choice earns value when it helps form a stable loop now. Rank it by its job, availability, current version and evidence—not by a timeless letter alone.

Early-game ranking filters

  • Can a new account realistically access and use it?
  • Does it solve clear speed, a hard target, or survival?
  • What gear, skills or Runes does it assume?
  • Was its effect checked for the active game version?
