GEAR REFERENCE

Task Bar Hero gear: upgrade the problem, not the item color.

Gear matters when it moves the next result: a cleaner farm, a passed boss check, or a stable stage. Treat every effect as version-sensitive until verified.

QUICK ANSWER

Keep an upgrade when it improves a measured outcome on the same route. An attractive effect that does not change the current bottleneck can wait.

Evaluate an item in context

  1. Name the stage or activity it should improve.
  2. Test before and after with the same build conditions.
  3. Check whether the new effect changes Rune or skill priorities.
  4. Record the game version alongside the result.
