CURRENT STATUS

No code is listed here without a current, verifiable official announcement and a clear redemption method. This is intentional.

How a code can be added

  • It must have a current official source or in-game confirmation.
  • The source date and redemption instructions must be visible.
  • Expiry or usage limits must be stated when known.
  • Expired and unverified codes remain off the active list.

Be careful with code searches

Code queries often attract copied lists and misleading downloads. Never install software, share account credentials, or use a third-party “generator” to redeem an in-game reward. A real redemption route should be available from the game or an official channel.
